# HPanel Infrastructure

This is the root of your PaaS system.

## Directory Structure

- `bin/` - Executable scripts (CLI tools)
- `templates/` - Configuration templates
- `services/` - Shared services configuration
- `projects/` - All application projects
- `logs/` - Centralized logs
- `ports.map` - Port allocation tracking
- `users.map` - User ownership tracking

## Usage

Scripts will be added in subsequent phases. Each script will be executable and well-documented.
