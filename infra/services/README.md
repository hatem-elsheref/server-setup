# Services Directory

This directory contains configuration for shared services.

## Structure

- `nginx/` - Nginx configuration
- `mysql/` - MySQL configuration
- `php/` - PHP-FPM configurations (multiple versions)
- `redis/` - Redis configuration
- `postgresql/` - PostgreSQL configuration
- `mongodb/` - MongoDB configuration
- `rabbitmq/` - RabbitMQ configuration
- `minio/` - MinIO configuration
- `postfix/` - Postfix (mail) configuration
- `dovecot/` - Dovecot (IMAP) configuration
- `vsftpd/` - FTP server configuration

## Note

Service configurations will be created during server initialization (Phase 3).
