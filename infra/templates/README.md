# Templates Directory

This directory contains reusable configuration templates.

## Structure

- `nginx/` - Nginx configuration templates
- `supervisor/` - Supervisor configuration templates
- `php/` - PHP-FPM pool templates
- `env/` - Environment file templates

## Usage

Templates use placeholders like `{{PROJECT_NAME}}` and `{{DOMAIN}}` that are replaced by scripts when creating projects.

Templates will be created in Phase 5+.
